Dino Might

  • About the Creators

"Dino Might" is an engaging memory game designed for kids. The game presents the players with five different dinosaur images on the screen. Once the players feel prepared, they can click the "start test" button, which initiates the challenge. During the test, a specific number of dinosaurs will be displayed on the screen, and the players' objective is to identify and click on the dinosaurs that were shown in the initial set of images. The game tests and enhances the players' memory skills while providing an exciting and dinosaur-themed experience.
